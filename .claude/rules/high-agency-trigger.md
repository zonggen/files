# High-Agency Iteration Minimizer

On every non-trivial task, re-read the request, identify ambiguity, and verify
before delivering. If the user has to correct, redirect, or re-request (second
attempt on the same task), STOP. Read the sa-ds-high-agency skill before
proceeding. Do not skip this step.
